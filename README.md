# 📰 NewsFlow - Real-time News Platform

A modern, responsive news aggregation platform built with **React**, **TypeScript**, and **Vite**. **NewsFlow** delivers up-to-the-minute global news with a focus on high-speed data fetching and a clean, user-centric interface.

📍 **Live Demo:** [news-flow-tau.vercel.app](https://news-flow-tau.vercel.app)  
📂 **GitHub Repo:** [github.com/bttu2002/NewsFlow](https://github.com/bttu2002/NewsFlow)

---

## 🔥 Key Features

- **Live News Updates:** Integrated **NewsAPI** with **Axios** to fetch and synchronize the latest global news articles instantly.
- **Advanced Search & Categorization:** Optimized search functionality allowing users to explore news by keywords and topics.
- **CORS Error Prevention:** Engineered an **Intermediate Backend (Proxy)** to handle browser-side security restrictions, ensuring 100% uptime during deployment.
- **Personalized Viewing:** Features a seamless **Dark/Light mode** toggle for an enhanced reading experience in any environment.
- **Universal Responsiveness:** Designed a flexible UI that adapts perfectly to Desktop, Tablet, and Mobile devices.

## 💡 Technical Highlights & Problem Solving

### 1. Robust API Integration with Axios
Instead of standard fetch, I utilized **Axios** for more robust error handling and interceptor capabilities. This allowed for smoother data fetching from **NewsAPI** and better management of asynchronous states across the application.

### 2. Overcoming Deployment Hurdles (CORS)
To address the common **CORS (Cross-Origin Resource Sharing)** issues found in client-side API requests, I implemented a **Backend Proxy Layer**. This demonstrates my ability to navigate technical constraints and provide production-ready solutions.

### 3. Type-Safe Content Management
By leveraging **TypeScript**, I created strict data models for articles and sources. This ensured that complex data structures from the API were handled predictably, significantly reducing bugs during the development process.
